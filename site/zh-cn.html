开源桌面 Markdown 编辑器

让 Markdown 写作安静下来。

Inkstill 把优雅写作、双向关联和丰富预览带到你电脑里的普通文件中。

直接下载安装版 免安装 ZIP 查看源码

Windows 可下载 · macOS 与 Linux 候选版 · MIT 开源许可

专注于写作本身

需要的都在，不需要的都不打扰。

打开单个文件，专心完成一篇文章；或打开整个文件夹，把笔记连接成知识工作区。

01

按自己的节奏写作

编辑、分栏和阅读视图，配合专注、打字机与海明威模式，适应不同写作状态。

02

把每个想法连接起来

搜索文件夹、跟随 Wiki 链接、查看反向链接并发现未链接提及，无需改变文件格式。

03

保留普通文件

内容仍是磁盘上的标准 Markdown，图片路径可迁移，无需注册账号就能开始写作。

键盘优先

整个工作区，一条命令就能到达。

在快速、键盘友好的面板中查找命令、格式、视图、导出和工作区操作。

  • 快速打开与全文搜索
  • 可搜索的文档大纲
  • 查找替换与表格构建器
  • 多标签页与会话恢复

丰富 Markdown

忠于源码地编辑，精致地输出。

在保持源码清晰、可迁移的同时，预览图表、数学公式、表格、脚注、代码块与提示框。

  • Mermaid 图表与 KaTeX 公式
  • GFM 表格、任务列表与脚注
  • 复制 HTML 与 HTML/PDF 导出
  • 本地图片导入与剪贴板粘贴

设计上安静，运行时克制。

Inkstill 围绕本地文件和有界后台任务构建。大文件保护、容量受控的工作区缓存与持续运行预算，让编辑器始终保持响应。
