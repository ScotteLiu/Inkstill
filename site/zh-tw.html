開源桌面 Markdown 編輯器

讓 Markdown 寫作安靜下來。

Inkstill 將優雅寫作、雙向連結與豐富預覽，帶進你電腦裡熟悉的普通檔案。

下載安裝版 免安裝 ZIP 查看原始碼

Windows 可下載 · macOS 與 Linux 候選版本 · MIT 開源授權

專注於工作本身

需要的都在，不需要的都不打擾。

開啟單一檔案專心完成文章，或開啟資料夾，把筆記連成知識工作區。

01

照自己的節奏寫作

編輯、分割與閱讀檢視，搭配專注、打字機及海明威模式，適合每一種寫作狀態。

02

連結每一個想法

搜尋資料夾、開啟 Wiki 連結、檢視反向連結與未連結提及，不必改變檔案格式。

03

保留普通檔案

內容仍是磁碟上的標準 Markdown，圖片路徑可攜，也無須註冊帳號。

鍵盤優先

整個工作區，一個命令就能抵達。

透過快速、鍵盤友善的面板尋找命令、格式、檢視、匯出與工作區操作。

  • 快速開啟與全文搜尋
  • 可搜尋的文件大綱
  • 尋找、取代與表格建立器
  • 多分頁與工作階段還原

豐富 Markdown

忠於原始碼地編輯，精緻地輸出。

在保持原始碼清楚、可攜的同時，預覽圖表、數學公式、表格、註腳、程式碼與提示框。

  • Mermaid 圖表與 KaTeX 公式
  • GFM 表格、工作清單與註腳
  • 複製 HTML 與 HTML/PDF 匯出
  • 本機圖片匯入與剪貼簿貼上

設計安靜，運作節制。

Inkstill 以本機檔案與受控背景工作為核心。大型檔案保護、容量受限的工作區快取與持續效能預算，讓編輯器保持流暢。
