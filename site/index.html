Open-source Markdown for desktop

A calmer way to write Markdown.

Inkstill brings beautiful writing, connected notes, and powerful preview tools to ordinary files on your computer.

Download installer Portable ZIP View source

Windows download · macOS and Linux candidates · MIT licensed

Made for the work itself

Everything you need. Nothing asking for attention.

Open one file for a focused draft, or a folder for an interconnected writing workspace.

01

Write in your rhythm

Edit, Split, and Read views pair with focus, typewriter, and Hemingway modes for the kind of session you need.

02

Connect every thought

Search folders, follow Wiki links, inspect backlinks, and find unlinked mentions without changing your file format.

03

Keep ordinary files

Your work remains standard Markdown on disk, with portable image paths and no account required to begin writing.

Keyboard first

Your whole workspace, one command away.

Search commands, formatting, views, exports, and workspace actions from a fast, keyboard-friendly palette.

  • Quick open and full-text search
  • Searchable document outline
  • Find, replace, and table builder
  • Multiple tabs and session restore

Rich Markdown

Source-faithful editing, polished output.

Preview diagrams, math, tables, footnotes, code, alerts, and more while the source stays readable and portable.

  • Mermaid diagrams and KaTeX math
  • GFM tables, tasks, and footnotes
  • Copy as HTML and HTML/PDF export
  • Local image import and paste

Quiet by design. Efficient by habit.

Inkstill is built around local files and bounded background work. Large-file safeguards, capped workspace caches, and continuous runtime budgets keep the editor responsive.
