# Inkstill 1.1.0 Preview

## English

Inkstill is a calm, local-first Markdown workspace for Windows x64. This preview combines source-faithful Markdown editing with tabs, workspace search, outlines, Wiki links, backlinks, recovery, safe previews, local image handling, and HTML/PDF export.

### Downloads

- **[Download Windows installer](https://github.com/ScotteLiu/Inkstill/releases/download/v1.1.0-preview.1/Inkstill-1.1.0.Setup.exe)** — `Inkstill-1.1.0.Setup.exe`
- **[Download portable ZIP](https://github.com/ScotteLiu/Inkstill/releases/download/v1.1.0-preview.1/Inkstill-win32-x64-1.1.0.zip)** — extract and run `Inkstill.exe`
- **[SHA-256 checksums](https://github.com/ScotteLiu/Inkstill/releases/download/v1.1.0-preview.1/SHA256SUMS.txt)** — verify both downloads

### Important preview notice

These binaries are not yet Authenticode-signed. Windows may show a SmartScreen warning. The source code, build pipeline, dependency lockfile, SBOM, third-party license inventory, release manifest, and checksums are published for inspection.

### Highlights

- Edit, synchronized Split, and Read views
- Tabs and last-session restoration
- Folder workspaces, full-text search, quick open, outline, backlinks, and unlinked mentions
- GFM, footnotes, KaTeX, Mermaid, Wiki links, TOC, YAML metadata, alerts, emoji, and sub/superscript
- Graphical Table Builder
- Focus, typewriter, and Hemingway modes
- Recovery journals, atomic writes, external-change conflict review, and EOL preservation
- Local image import and clipboard paste
- HTML/PDF export and Copy as HTML
