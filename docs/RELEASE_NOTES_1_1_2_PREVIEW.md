# Inkstill 1.1.2 Preview

## English

This preview refines the workspace layout after feedback on 1.1.1.

### Highlights

- Floats the workspace panel with an even margin on every side, removing the
  full-height divider line that previously sat against the sidebar.
- Keeps the same rounded panel styling at compact window sizes.

### Updating

Installed Windows builds can update through **Help → Check for Updates…**;
the installer is verified against its SHA-256 checksum before it runs.
Portable builds can check for updates but will switch to the installed
edition when the installer runs.

These preview binaries are not yet Authenticode-signed, so Windows may display
a SmartScreen warning.

## 简体中文

此预览版根据 1.1.1 的反馈改进了工作区布局。

### 主要改进

- 工作区面板四周保持一致的留白，去掉了原来紧贴侧栏的一条通高分隔线。
- 小窗口下保持同样的圆角面板样式。

### 更新方式

Windows 安装版可通过 **Help → Check for Updates…** 更新，
安装器运行前会先完成 SHA-256 校验。
免安装版同样可以检查更新，但运行更新安装器后会切换为安装版。

当前预览版尚未进行 Authenticode 签名，因此 Windows 可能显示 SmartScreen 警告。
